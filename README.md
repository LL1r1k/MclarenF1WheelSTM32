# Реплика гоночного руля F1 McLaren
Напечатанный на 3Д принтере корпус руля, начинка STM32, работающие кнопочки и LCD экран.
 
## Требования
- STM32F103
- LCD Monitor
- Кнопки
- Потенциометр

## Использование
Открыть проект в STM32CubeIDE
